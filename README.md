# chatpc

#### 一款基于electron开发的桌面聊天应用

``` bash
# 如果你没有vue-cli的话需要全局安装
npm install -g vue-cli

# 然后使用vue-cli来安装electron-vue的模板
vue init simulatedgreg/electron-vue my-project

# 安装依赖
cd my-project
npm install # or yarn

# 进入开发模式
npm run dev # or yarn run dev


```
